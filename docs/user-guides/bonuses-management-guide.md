# Guía de Gestión de Bonificaciones

## Visión General

El módulo de **Bonificaciones** administra los incentivos, premios y reembolsos del sistema. Permite registrar y consultar todos los conceptos de compensación.

## Funcionalidades Principales

- **Registrar** bonificaciones (Bonificación, Incentivo, Premio, Reembolso)
- **Consultar** historial de bonificaciones
- **Filtrar** por tipo y fecha
- **Exportar** reportes de bonificaciones

## Cómo Usar

1. Accede al módulo **Bonificaciones** desde el menú principal
2. Haz clic en **Nueva Bonificación** para registrar
3. Completa los campos: tipo, descripción, monto y fecha
4. Guarda la bonificación
5. Usa los filtros para consultar bonificaciones existentes
