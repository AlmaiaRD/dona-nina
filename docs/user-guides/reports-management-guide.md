# Guía de Gestión de Reportes

## Visión General

El módulo de **Reportes** permite generar informes financieros y operativos para analizar el rendimiento del negocio.

## Tipos de Reportes

- **Ventas:** Facturación por período
- **Cobros:** Recibos y métodos de pago
- **Inventario:** Valor del stock actual
- **Clientes:** Compras por cliente
- **PV:** Puntos de Volumen generados
- **Gastos:** Gastos registrados

## Cómo Usar

1. Accede al módulo **Reportes** desde el menú principal
2. Selecciona el tipo de reporte que deseas generar
3. Configura el rango de fechas (opcional)
4. Haz clic en **Generar Reporte**
5. Revisa los datos en pantalla
6. Haz clic en **Exportar** para descargar en Excel
