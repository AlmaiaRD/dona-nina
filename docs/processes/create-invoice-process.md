# Proceso: Cómo Crear una Factura

## Paso 1: Acceder al Módulo
1. Haz clic en **Facturación** en el menú principal
2. Haz clic en el botón **Nueva Factura**

## Paso 2: Seleccionar Cliente
1. Busca el cliente por nombre o teléfono
2. Selecciona el cliente de la lista
3. Si el cliente no existe, créalo primero

## Paso 3: Agregar Productos
1. Busca productos por nombre o código
2. Selecciona el producto
3. Ingresa la cantidad
4. Ajusta el precio si es necesario

## Paso 4: Configurar la Factura
1. Selecciona el margen (30% o 35%)
2. Agrega notas si es necesario
3. Revisa los totales

## Paso 5: Guardar y Emitir
1. Haz clic en **Guardar**
2. La factura se genera con estado Pendiente
3. Puedes descargar el PDF o imprimir
