# Guía de Gestión de Documentos

## Visión General

El módulo de **Documentos** centraliza todos los documentos comerciales en un solo lugar: facturas, recibos y compras. Puedes buscar, filtrar y descargar cualquier documento desde aquí.

## Funcionalidades Principales

- **Visualizar** todas las facturas, recibos y compras
- **Buscar** documentos por número, cliente o proveedor
- **Filtrar** por tipo de documento
- **Descargar** documentos en PDF o JPG
- **Ver detalles** de cada documento

## Cómo Usar

1. Accede al módulo **Documentos** desde el menú principal
2. Usa los filtros rápidos (Facturas, Recibos, Compras, Todos)
3. Escribe en la barra de búsqueda para encontrar documentos específicos
4. Haz clic en el ojo (👁️) para ver el detalle
5. Haz clic en la flecha (⬇️) para descargar
