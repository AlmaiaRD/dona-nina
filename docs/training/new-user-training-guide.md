# Guía de Capacitación para Nuevos Usuarios

## Bienvenido a Almaia RD

Esta guía te ayudará a familiarizarte con el sistema en tu primera semana.

## Día 1: Introducción
1. Inicia sesión con tus credenciales
2. Explora el menú principal
3. Visita el Dashboard para ver el resumen

## Día 2: Facturación
1. Aprende a crear una factura
2. Practica agregar productos
3. Revisa cómo se calculan los totales

## Día 3: Clientes
1. Agrega un cliente nuevo
2. Busca clientes existentes
3. Revisa el historial de un cliente

## Día 4: Inventario
1. Revisa el stock actual
2. Aprende a registrar una compra
3. Consulta los movimientos de inventario

## Día 5: Reportes
1. Genera un reporte de ventas
2. Exporta datos a Excel
3. Revisa las estadísticas del dashboard

## Recursos Adicionales
- Consulta las guías de usuario para cada módulo
- Usa el chat de soporte para ayuda rápida
- Revisa los tutoriales en Almaia Academy
