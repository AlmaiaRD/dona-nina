# Guía de Gestión de Créditos

## Visión General

El módulo de **Créditos** gestiona los créditos y saldos a favor de los clientes. Permite controlar cuentas por cobrar y administrar abonos.

## Funcionalidades Principales

- **Ver saldos** de créditos disponibles por cliente
- **Registrar abonos** a facturas pendientes
- **Consultar historial** de créditos usados
- **Gestionar estados** de créditos (Disponible, Usado, Expirado)

## Cómo Usar

1. Accede al módulo **Créditos** desde el menú principal
2. Revisa la lista de créditos disponibles
3. Usa el buscador para encontrar créditos por cliente
4. Haz clic en un crédito para ver su detalle
5. Usa el botón **Usar Crédito** para aplicarlo a una factura
